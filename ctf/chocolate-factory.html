Chocolate Factory CTF

Overview

The Chocolate Factory machine is a web-based vulnerable target designed to simulate a real-world attack chain starting from web exploitation and ending with full system compromise. The objective was to identify weaknesses in exposed services, exploit insecure functionality to obtain shell access, and escalate privileges through Linux misconfigurations.

Enumeration & Recon

  • Conducted network scanning to identify open services and exposed web ports
  • Discovered hidden directories and endpoints through brute-force enumeration
  • Identified vulnerable parameters within web functionality
  • Mapped application behavior to locate potential injection points

Initial Exploitation

  • Exploited unsanitized user input leading to command execution
  • Crafted payloads to establish reverse shell access
  • Successfully obtained low-privileged access to the target Linux environment

Post-Exploitation

  • Performed manual Linux enumeration for privilege escalation vectors
  • Identified sensitive files and configuration weaknesses
  • Discovered insecure permissions and exploitable local paths

Privilege Escalation

  • Leveraged misconfigured binaries and permission flaws
  • Escalated privileges from limited shell to root user
  • Achieved full administrative control over the system

Key Skills Demonstrated

  • Web vulnerability discovery
  • Reverse shell handling
  • Linux enumeration techniques
  • Privilege escalation methodology
  • Structured reporting & analysis
