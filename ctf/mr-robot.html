Mr Robot CTF

Overview

Mr Robot is a WordPress-focused CTF simulating common real-world web exploitation scenarios. The challenge involved gaining access through web enumeration and credential abuse, establishing shell access via file upload vulnerabilities, and performing privilege escalation on a Linux system.

Enumeration

  • Performed Nmap scanning to identify active services
  • Enumerated WordPress directories, plugins, and exposed files
  • Discovered login portal and brute-force targets
  • Located sensitive files revealing usernames

Exploitation

  • Conducted credential attacks against WordPress authentication
  • Gained administrator dashboard access
  • Uploaded malicious PHP reverse shell via theme editor
  • Triggered payload to obtain remote shell

Local Enumeration

  • Extracted local user credentials
  • Pivoted between users for higher privileges
  • Identified SUID binaries and misconfigurations

Privilege Escalation

  • Exploited vulnerable SUID binary
  • Escalated privileges to root
  • Achieved full system compromise

Key Skills Demonstrated

  • WordPress enumeration & exploitation
  • Credential attacks
  • Reverse shell deployment
  • Linux privilege escalation
  • Attack chain methodology
